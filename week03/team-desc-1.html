CSS Box Shadow
The box-shadow CSS property creates a shadow behind virtually any HTML element.
According to the MDN documentation, the following possible configurations are possible:
› offset-x | offset-y | color
› offset-x | offset-y | blur-radius | color
› offset-x | offset-y | blur-radius | spread-radius | color
› inset    | offset-x | offset-y    | color
In the following example, any <div> element will be given a blurred, dark-blue shadow to its lower-right:
div {
  box-shadow: 10px 10px 5px #007;
}
