CSS Text Shadow
The text-shadow property places a shadow behind the text of a given element.

According to the MDN documentation, the following possible configurations are possible:
› offset-x | offset-y | blur-radius | color
› color    | offset-x | offset-y    | blur-radius
› offset-x | offset-y | color
› color    | offset-x | offset-y
› offset-x | offset-y
In the following example, the text within any <p> element will be given a blurred, medium-blue shadow to its lower-right:
p {
  text-shadow: 1px 1px 1px #66f;
}
