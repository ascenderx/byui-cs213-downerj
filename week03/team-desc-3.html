CSS Overflow
The overflow property clips the content within an element based on the current dimensions, adding optional behavior such as scrolling.

According to the MDN documentation, the following possible configurations are possible:
› visible
› hidden
› scroll
› auto
› hidden
› visible
In the following example, the browser will automatically clip the text in any <div> element, which in most cases will cause vertical scrolling:
div {
  overflow: auto;
}
