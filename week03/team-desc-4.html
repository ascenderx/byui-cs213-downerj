CSS Maximum Width
The max-width property instructs an element to restrict its width to within a specified (or computed) value. This is used in creating responsive webpages so that certain elements that would otherwise be very wide will instead fit inside of a mobile viewport.

According to the MDN documentation, the following possible configurations are possible:
› <length value>
› <percent value>
› none
› max-content
› min-content
› fit-content
› fill-available
In the following example, the browser will expand every <div> with the class constrained to no wider than 1000 pixels.
div.constrained {
  max-width: 1000px;
}
