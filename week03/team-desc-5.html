CSS Media Queries
The @media() directive instructs the browser to render certain elements with a different style given a certain condition of the viewport. For example, the browser can detect when a screen is a certain size and render the CSS differently based on the media query specified by the web page's stylesheet(s).

According to the MDN documentation, there are numerous keywords and values that are considered valid syntax for media queries. Some examples that given at MDN are:
› @media print            /* used for printers */
› @media screen           /* used for mobile & desktop screens */
› @media screen, print    /* used for screens and for printing */
› @media (width <= 700px) /* used for any viewport less than 700px wide */
The code that follows the query signature consists of the new styles to applied.

In the following example, the browser will render every <div> of class big-block with a width of 500 pixels unless the viewport's width is 700 pixels or less, in which case it will ensure that the block's width takes up 95% of the containing element's width:
div.big-block {
  width: 500px;
}

@media (max-width: 700px) {
  div.big-block {
    width: 95%;
  }
}
